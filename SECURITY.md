# Security Policy

## Supported versions

This project currently supports the latest `0.2.x` line.

## Reporting a vulnerability

Please do **not** open public issues for security vulnerabilities.

Send a report to `security@ace-lite.dev` with:

- A clear description of the issue
- Reproduction steps or proof of concept
- Affected version/commit
- Any suggested remediation

We will acknowledge receipt within 72 hours and provide a remediation plan.

## Disclosure policy

- Coordinate disclosure with maintainers.
- After a fix is available, we will document impact and mitigation.
