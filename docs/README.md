# Docs (ACE-Lite Engine)

This folder keeps the public documentation surface compact: a small user entry
path, a handful of stable reference docs, and a narrower set of maintainer
runbooks.

## Start here

- `docs/guides/GETTING_STARTED.md`
- `docs/guides/MCP_SETUP.md`
- `docs/design/ARCHITECTURE_OVERVIEW.md`
- `docs/maintainers/README.md`
- `docs/maintainers/SYSTEM_MAP.md`
- `docs/reference/ACEIGNORE_COOKBOOK.md`

## Layout

- `docs/guides/` - the shortest path to a working local or MCP setup
- `docs/design/` - durable architecture notes, not iteration logs
- `docs/reference/` - stable reference docs such as ignore rules and upgrade notes
- `docs/maintainers/` - release, benchmark, and quality runbooks for maintainers
- `docs/case_studies/` - optional usage examples
