# CLI Cheatsheet

Common entrypoints:

- `ace-lite plan --query="..." --repo=<repo> --root=.`
- `ace-lite plan --quick --query="..." --repo=<repo> --root=.`
- `ace-lite index --root=.`
- `ace-lite doctor --root=. --fast`
- `ace-lite benchmark run --cases benchmark/cases/default.yaml --root=.`
